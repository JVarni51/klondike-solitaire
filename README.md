# Klondike Solitaire

Play free at **https://jvarni51.github.io/klondike-solitaire/**

A browser-based Klondike Solitaire game. No build step, no backend.

## Features

- Classic Klondike rules (7 tableau piles, 4 foundations)
- Drag-and-drop card movement
- Double-click to auto-move to foundation
- Draw 1 or Draw 3 stock modes
- Timer, move counter, hints
- Win detection with play-again overlay
- Responsive layout for desktop and mobile

## Local Development

ES modules require a local server:

```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1
```

Then open http://localhost:8080
